# AI Appointment Booking Receptionist

An AI-powered appointment booking system built using n8n, Tally Forms, Gmail, and OpenRouter AI.

## Features

* Smart Appointment Booking Form
* AI Appointment Processing
* Automated Client Confirmation Emails
* Admin Notification Emails
* Webhook-Based Workflow
* No-Code Automation with n8n

## Workflow

1. User submits appointment form
2. Tally sends data to n8n webhook
3. AI processes appointment request
4. System extracts appointment details
5. Confirmation email sent to client
6. Notification email sent to admin

## Tech Stack

* n8n
* Tally Forms
* Gmail
* OpenRouter AI
* Webhooks
